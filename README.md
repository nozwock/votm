## Specific Dependencies
  * tkinter
  * tabulate
  * pycrypto
  
## Usage
 * Clone the repo, and then
 ```sh
 $ cd votm
 $ pip install -r requirements.txt
 ```
 * For Edit App
 ```sh
 $ python votm_vte.py
 ```
 * For Vote App
 ```sh
 $ python votm_edt.py
 ```
